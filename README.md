# Character Recognition

## Project Stack
* Java v17
* Sprint Boot v2.6.3
* Python v3.8
* Fast API
* Node.js v18.12.1
* React.js v18.2.0
* Neo4j
* Docker

## Start Project Locally

1. Create [.env](.env) file locally.
2. Copy all the values from the [.env.tpl](.env.tpl) to the [.env](.env).
3. Run from the terminal:

```shell
docker-compose up
```
